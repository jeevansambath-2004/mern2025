E-court Case Management Portal

A comprehensive web platform to digitize and streamline court case records for clients and lawyers. Built with the MERN stack (MongoDB, Express.js, React, Node.js), it features secure document uploads, real-time case tracking, and transparent communication.

  • User authentication and role-based dashboards for lawyers and clients (JWT-based).
  • Secure file storage on AWS S3 with encryption for sensitive documents.
  • Real-time notifications for hearings and updates using Socket.io.
  • Advanced search/filter for quick access to case histories and documents.
  • Audit logs for all actions and admin panel for user/permission management.
  • Responsive UI with React and Material-UI.
  • Planned: Integration with government APIs for e-filing and AI-based document analysis.

Subscription & Payment Manager

A web application to track and manage online subscriptions and recurring payments. Backend is Python (Flask), frontend is React, and MongoDB is used for data storage.

  • Automated email/SMS reminders for renewals and failed payments (Twilio, SendGrid APIs).
  • Interactive dashboard with analytics and charts (Chart.js) for spending trends.
  • Secure login, password hashing, and JWT-based sessions.
  • Mobile-friendly design with PWA support for offline use.
  • Export options for subscription/payment history (CSV, PDF).
  • Planned: Payment gateway integration, AI-based spending recommendations, multi-user support.

College Admission Dashboard

A CRUD web application built with Flask and SQLite for college staff to manage student admission records.

  • REST API backend (Flask) and simple HTML/CSS/JS frontend (Bootstrap for responsive design).
  • Search, filter, and export (CSV/PDF) of student records.
  • User authentication with admin/staff roles.
  • Bulk import/export of student data for efficient onboarding.
  • Form validation and error handling for data integrity.
  • Deployed on Heroku, Docker support for local development.
  • Planned: SMS/email notifications, analytics dashboard, PostgreSQL migration.
