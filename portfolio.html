Projects

E-court Case Management Portal

A comprehensive web platform designed to digitize and streamline the management of court case records for both clients and lawyers. The portal enables secure document uploads, real-time case tracking, and transparent communication between legal professionals and their clients.

  • Built with MERN stack (MongoDB, Express.js, React, Node.js) for a robust, scalable architecture.
  • User authentication and authorization using JWT, with role-based dashboards for lawyers and clients.
  • Secure file storage and retrieval using AWS S3, with encryption for sensitive documents.
  • Real-time notifications and updates for hearings, filings, and case status via Socket.io.
  • Advanced search and filtering for quick access to case histories and documents.
  • Audit logs for all actions to ensure traceability and accountability.
  • Responsive UI for both desktop and mobile users, built with React and Material-UI.
  • Admin panel for managing users, permissions, and system settings.
  • Future enhancements: Integration with government APIs for e-filing, calendar sync, and AI-based document analysis.
Subscription and Payment Manager

A user-friendly web application that helps individuals and businesses efficiently track and manage their online subscriptions and recurring payments.

  • Developed using Python (Flask) for backend REST APIs and React for frontend SPA.
  • MongoDB for storing user, subscription, and payment data, with aggregation for analytics.
  • Automated email/SMS reminders using third-party APIs (e.g., Twilio, SendGrid) for upcoming renewals and failed payments.
  • Interactive dashboard with charts and analytics (Chart.js) for visualizing spending trends and upcoming expenses.
  • Secure login and user management with password hashing and JWT-based sessions.
  • Mobile-friendly design for easy access on any device, with PWA support for offline use.
  • Export options for subscription and payment history (CSV, PDF).
  • Future enhancements: Integration with payment gateways for direct payment, AI-based spending recommendations, and multi-user support for families/teams.
College Admission Dashboard

A CRUD (Create, Read, Update, Delete) web application built with Flask and SQLite, tailored for college staff to manage student admission records.

  • Backend: Python Flask REST API with SQLite database for lightweight, easy deployment.
  • Frontend: Simple HTML/CSS/JavaScript interface for staff users, with Bootstrap for responsive design.
  • Features search, filter, and export (CSV/PDF) of student records for reporting and analysis.
  • User authentication for secure access, with different roles for admin and staff.
  • Form validation and error handling for data integrity and user feedback.
  • Bulk import/export of student data for efficient onboarding and record-keeping.
  • Deployed on Heroku for demonstration and testing, with Docker support for local development.
  • Future enhancements: Integration with SMS/email APIs for admission notifications, analytics dashboard for admission trends, and migration to PostgreSQL for scalability.
